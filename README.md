# Fortura

Fortura is an AI-powered finance management and budgeting app built using React Native and Expo.

## Installation and Setup

Follow these steps to install and run the project:

### Prerequisites
- Node.js installed (latest LTS recommended)
- Expo CLI installed globally (`npm install -g expo-cli`)
- Physical device with Expo Go installed

### Steps to Run the Project

1. Clone the repository (only this branch):
   ```sh
   git clone --single-branch --branch main https://github.com/yourusername/fortura.git
   cd fortura
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Start the Expo development server:
   ```sh
   npx expo start
   ```

4. Run on a device:
   - **Android:** Scan the QR code with Expo Go
   - **iOS:** Scan QR with Expo Go

Now, you can start using Fortura for finance management and budgeting!
